GameObject Arguments

X Coord: -x (horizontal location)

Y position: -y (vertical location)

Vertex: -v (x offset) (y offset)

Tag: -t (tag)

Update Function: -u (update function name)

Late Update Function: -lu (late update function name)

Collide Function: -cf (collide function name)

Animations: -a (animation package name)

Current Animation: -ca (index of animation to start with)

Default Frame: -df (path to default frame location)

Default Length: -dl (default length value)

Default Width: -dw (default width value)

Debug Mode: -d

Property: -p (property name) (property value)

Object Property: -op (property name) (variable name of property value)

Canvas: -cv (Background/Play/Foreground/Grid/UserInterface)

Audio: -au (path to audio file)

Init function: -i (init function name)
